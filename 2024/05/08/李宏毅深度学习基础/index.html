三、深度学习基础

当神经网络的梯度变为0时，此时可能位于局部极大或者极小值，但也可能位于鞍点处，为了跳出鞍点，进行如下分析。

判断临界值种类的方法：根据泰勒近似，，其中为海森矩阵。当一阶导为零时，如果正定，说明 ，表明这是一个局部极小值；如果负定，则为局部最大值；否则其为鞍点。

自适应学习率：如果在某一个方向上，梯度的值很小，非常平坦，我们会希望学习率调大一点；如果在某一个方向上非常陡峭，坡度很大，我们会希望学习率可以设得小一点。

  • 典型方法为 AdaGrad： 对于普通的梯度更新 ，这里的 是固定参数，可以将乘以一个动态权重 ，这里常将 设置为梯度的均方根。
  • Adam 优化器 = RMSprop + 动量法 ：其中RMSprop可以理解为AdaGrad的改进，它对权重进行改进为历史所有梯度的带权重之和，能够更快反应，“踩油门跟踩刹车”；动量指的是每次在移动参数的时候，不是只往梯度的反方向来移动参数，而是根据梯度的反方向加上前一步移动的方向决定移动方向，一定程度上能够跳出局部极小值。

五、自监督学习

BERT模型是自监督领域常见模型，架构与Transformer的编码器完全一致，输入向量与输出向量长度一致。

如何使用 BERT？在训练时，让 BERT 学习两个任务.

  • 把一些字符掩盖起来，让它做填空题，补充掩码的字符
  • 预测两个句子是否有顺序关系（两个句子是否应该接在一起），这个任务没什么用

下游任务：通用语言理解评估

  1. 情感分析：给机器一个句子，并告诉它判断句子是正面的还是负面的。输入BERT模型只要在输入向量前面加一个[CLS]词元，经过BERT网络，对 [CLS] 对应的向量应用线性变换，经过softmax来进行分类
  2. 词性标注：BERT模型保持不变，对输出的向量进行线性变化，然后进行训练
  3. 自然语言推理（如判断评论对于文章的立场）：给定两个句子，这两个句子之间有一个特殊的分隔词元 [SEP]，并把 [CLS] 词元放在最前面的位置。 这个序列是 BERT 的输入，然后BERT 将输出另一个长度与输出长度相同的序列。 但只将 [CLS] 词元作为线性变换的输入，然后决定输入这两个句子，输出应该是什么类别
  4. 基于提取的问答：在此任务中，输入序列包含一篇文章和一个问题. 文章和问题都是一个序列，, ，问题和文章之间有一个特殊标记 [SEP]
